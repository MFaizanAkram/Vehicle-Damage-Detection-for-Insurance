# Vehicle Damage Detection for Insurance

A deep learning project that classifies vehicle damage severity from images to automate insurance claim processing.

---

## Damage Classes

| Class     | Coverage Estimate      |
|-----------|------------------------|
| No Damage | PKR 0 — Claim Rejected |
| Minor     | PKR 15,000 – 50,000    |
| Moderate  | PKR 50,000 – 1,50,000  |
| Severe    | PKR 1,50,000+          |

---

## Tech Stack

- Python, TensorFlow, Keras
- VGG16 Transfer Learning
- Streamlit

---

## Run the App

```bash
pip install streamlit tensorflow pillow numpy
streamlit run app.py
```
